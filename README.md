# Job Portal Project

This repository contains a job portal application with separate frontend and backend services.

## Live Deployment
- Frontend (Netlify): https://jobportal-yt-main.netlify.app
- Backend (Render): https://jobportal-yt-main-nv35.onrender.com

## Local Setup
### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
